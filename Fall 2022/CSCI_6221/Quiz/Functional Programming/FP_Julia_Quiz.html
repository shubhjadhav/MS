High-order Functions in Julia

Julia supports high-order functions similar to python. In Julia, high-order functions have same syntax as first-class functions. If we do not define any data type of parameter, by default Julia considers it as "Any" data type and we can pass any input to the function. Julia also has several in-build high-order function for implementation such as, filter() and map(). The following code acts as an example for implementation of high-order functions in Julia. 



---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Case 1: If a function takes one or more functions as its parameters
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

#Define the (High-order) function to compare two functions for single value to return a string

MaxFunction(fn1::Function, fn2::Function, val) = fn1(val) > fn2(val) ? "Exponential Function is larger " : "Polynomial Function is larger"

#Define two functions describing mathematical equations

ExponentialFunc(x) = 2^x

PolynomialFunc(x) = x^100

#Call the compare function

MaxFunction( ExponentialFunc, PolynomialFunc, 12)

MaxFunction( ExponentialFunc, PolynomialFunc, 100)

Output for 1st call: "Exponential Function is larger"
Output for 2nd call: "Polynomial Function is larger"

Alternately, we can use the exact code from the example provided in the question

f(x) = x + 2

g(x) = 3 *x

b(x) = f(g(x))

#Make call
b(10)

Output: 32


---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Case 2: If a function returns a function as its result
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

#Define a function to convert string in upper case
ToUpper(val::String) = return uppercase(val)

#Call the Function
ToUpper("Programming is Fun!")

Output: "PROGRAMMING IS FUN!"
